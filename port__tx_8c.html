◆ _timer_pwm_setup()

static void _timer_pwm_setup ( uint32_t  tx_id)
static

Configure the PWM timer. This timer configures the PWM for the NEC protocol.

TO-DO alumnos:
PWM timer setup

✅ 1. Enable the clock source of our timer
✅ 2. (Optional) Disable the counter (register CR1) and enable the autoreload preaload (register ARPE)
✅ 3. Reset the timer count (register CNT)
✅ 4. Set the value of the autoreload register (register ARR)
✅ 5. Set the value of the prescaler (register PSC)
✅ 6. Generate an update event (register EGR)

PWM mode configuration

✅ 7. Disable the output compare of the corresponding channel (register CCER)
✅ 8. Set mode PWM 1, and enable preload (register CCMR)
✅ 9. Set the pulse width of the corresponding channel (register CCRx)

Parameters
tx_idTransmitter ID. If we have more than one transmitter, this value allows us to know which one of the we want to configure.

Configure the symbol timer. This timer sets the tick base as a reference for the symbols of the protocol.

TO-DO alumnos:

✅ 1. Enable the clock source of our timer
✅ 2. (Optional) Disable the counter (register CR1) and enable the autoreload preaload (register ARPE)
✅ 3. Reset the timer count (register CNT)
✅ 4. Set the value of the autoreload register (register ARR)
✅ 5. Set the value of the prescaler (register PSC)
✅ 6. Generate an update event (register EGR)
✅ 7. Clean the interrupt flag of the status register (register SR)
✅ 8. Enable interrupts (register DIER)

Eventually sets the priority and enable interruptions globally.

◆ port_tx_init()

void port_tx_init ( uint8_t  tx_id,
bool  status 
)

Configure the HW specifications of a given infrared transmitter.

TO-DO alumnos:

✅ 1. Call function port_system_gpio_config() with the right arguments to configure the infrared transmitter GPIO to act with alternate function, and no pull up neither pull down connection
✅ 2. Call function port_gpio_alternate_config() with the right arguments to configure the GPIO as PWM.
✅ 3. Call function _timer_symbol_setup() to configure the symbol timer. This timer is the base tick reference for all the infrared transmitters that system could have
✅ 4. Call function _timer_pwm_setup() to configure the PWM timer of the given infrared transmitter
✅ 5. Call function port_tx_pwm_timer_set() to set the PWM ON or OFF according to the given status

Parameters
tx_idTransmitter ID. This index is used to select the element of the transmitters_arr[] array
statusTo indicate if PWM starts, or not, from the beginning
